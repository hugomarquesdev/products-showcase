# Products Showcase

Website / Webapp designed to showcase a variety of products using Gatsby.

- Slideshow
- Scroll Animations

- Image Markers

## Technologies Used

- React
- Gatsby
- styled-components
- i18next
- Redux

## Installation

```bash
1. Install dependencies (`yarn install `)
2. `yarn develop`
